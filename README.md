## Inicializar Projeto

Primeiro devemos adicionar os node_modules:

npm install

ou 

yarn install

Agora para incializar o projeto basta rodar o script de start:

npm start

ou 

yarn start


### Vizualizando o resultado

No seu navegador, acesse [http://localhost:3000](http://localhost:3000).<br/>
